# Расписание твоей любимки — лето 2026

Статическая HTML-страница с расписаниями Димы и любимки.

## Возможности

- текущее время в Томске (`Asia/Tomsk`, UTC+7);
- переключение между расписаниями Димы и любимки;
- шкала полных суток от `00:00` до `24:00`;
- автоматическое определение текущего события;
- подсветка текущего занятия непосредственно в расписании;
- пастельно-розовая тема свиданий и романтичные подписи в свободных окнах;
- счетчики времени общения, первой встречи и первого поцелуя;
- выбор текущего пользователя: Дима или Александра;
- общие желания встретиться с произвольным комментарием в свободных окнах;
- горизонтальная прокрутка по 24-часовой шкале;
- адаптивный вид для компьютера и телефона;
- фильтры прошедших дней и дней без занятий;
- справочник дисциплин и преподавателей.

## Локальный запуск

```powershell
python server.py
```

После запуска откройте `http://localhost:8080`.

Отметки сохраняются в SQLite-файле `data/requests.sqlite3`.

Для проверки произвольного момента по томскому времени можно использовать параметр `now`:

```text
http://localhost:8080/?now=2026-06-19T17:30:00
```
